Anomalias em Jobs do Rundeck

Este site publica o artefato anomalies.json gerado por uma esteira MLOps (n8n → GitHub Actions → AWS Amplify).

Se o botão não abrir, tente manualmente: /anomalies.json ou /app/anomalies.json.

Como funciona a esteira (resumo)

  • Coleta (n8n): Cron → HTTP Request no Rundeck /api/38/executions → map → append em data/dados_rundeck.csv no GitHub.
  • Detecção (GitHub Actions): ao receber push, roda scripts/pipeline.py (ETL + features). Usa Z-score por job e global (e opcionalmente IsolationForest) para gerar app/anomalies.json.
  • Publicação (Amplify): faz deploy da pasta app/; regra de SPA mantém index.html. Regra explícita libera /anomalies.json sem rewrite.

Variáveis principais: RUNDECK_BASE_URL, RUNDECK_API_TOKEN, RUNDECK_PROJECT, ANOMALY_THRESHOLD.
